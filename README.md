MediaPortalIptvChannelsPluginDVBI is DVB-I provider plugin for MediaPortalIptvChannels plugin.

Main features:
- DVB-I client: search for available Service Lists (in Central Service Registry or Multicast DNS), automatically create corresponding MediaPortal TV Channels,
                automatic/manual service/region/LCN selection based on user's location configuration, Content Guid (EPG) supported
- DVB-I server: create own Service Lists from MediaPortal TV Server Channels, Content Guid (EPG) is supported (content is taken from TV Server channel), Multicast DNS support
